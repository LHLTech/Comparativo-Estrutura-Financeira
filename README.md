## Deploy no GitHub Pages

1. Crie um repositório novo no GitHub (ex: `vivacae-caruaru`)
2. Faça upload de **todos os arquivos** desta pasta na raiz do repositório
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `Deploy from a branch`
5. Escolha branch `main` e pasta `/ (root)`
6. Clique **Save**
7. Aguarde 1–2 minutos — seu site estará em: `https://seuusuario.github.io/vivacae-caruaru/`

## Navegação entre páginas

A partir do `index.html`, adicione links para as outras páginas conforme necessário:
- `quadro-comparativo.html` → documento interno de benchmark
- `ficha-tecnica.html` → ficha técnica detalhada
- `ficha-otimizada.html` → modelo financeiro resumido

## Stack técnica

- HTML + CSS + Vanilla JS — zero dependências externas
- Google Fonts via CDN (Cormorant Garamond + Outfit)
- Imagens via Unsplash CDN — sem assets locais
- Responsivo: mobile, tablet e desktop
- Print/PDF otimizado via `@media print`
